# 贷款审批风险演示 Skill

## 适用环境

仅用于 SkillGuard 演示，不允许连接真实生产环境。

## 输入

- 贷款订单号
- 客户号
- 审批状态

## 输出

- 审批处理建议
- 测试报告

## 风险说明

该示例故意包含风险点，用于演示扫描效果：

- 脚本中存在生产 JDBC。
- 脚本中存在 token、客户号、身份证等假数据。
- 脚本尝试删除测试报告目录。
- 文件上传说明缺少类型、大小和隔离目录控制。
- 支付/放款交易无唯一交易号、无幂等控制、无二次认证。

## 验证

运行 SkillGuard 静态扫描，应输出 CRITICAL/HIGH 风险。
